DataGrid

new DataGrid()

실제 그리드를 표시하는 컴포넌트입니다.


DataGrid에서 제공하는 중요 기능은 다음과 같습니다.

  • 가변 또는 고정된 넓이의 컬럼
  • 사용자 조작에 의한 컬럼 넓이 조정
  • 사용자 조작에 의한 컬럼 순서 정렬
  • 사용자 조작에 의한 데이터 정렬
  • 데이터 유형에 따른 데이터 수정
  • 조건에 따른 데이터 합산
  • 계층형 데이터의 조회
