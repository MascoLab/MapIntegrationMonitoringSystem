Class: Widget

Widget

new Widget()

Widget은 대시보드의 구성요소 중 하나로 component를 담는 컨테이너입니다.

Members

minHeightnumber

위젯의 최소 높이

Default Value:
  • 10
See:
  • 자바스크립트에서 사용할 때는 getter, setter함수를 사용합니다. - getMinHeight(), setMinHeight(val)

minWidthnumber

위젯의 최소 너비

Default Value:
  • 10
See:
  • 자바스크립트에서 사용할 때는 getter, setter함수를 사용합니다. - getMinWidth(), setMinWidth(value)

Methods

destroy()

widget 삭제시 이벤트 리스너 제거 및 초기화하는 함수입니다.

getSlide(){Slide}

해당 위젯이 속해있는 슬라이드의 인스턴스를 얻습니다.

Returns:
Type Description
Slide 해당 위젯이 속해있는 슬라이드의 인스턴스

setBorderColor(rgb)

위젯의 Border 색상을 설정합니다

Name Type Description
rgb string

또는 hex

setBorderWidth(Border)

위젯의 Border 굵기를 설정합니다

Name Type Description
Border string

굵기(px단위)

showMaximized()

위젯을 슬라이드 크기만큼 리사이징 합니다.

showNormal()

위젯을 이전 사이즈로 리사이징 합니다.

Events

activeHandler

위젯이 소속된 슬라이드가 활성화되면 호출됩니다.

widget.activeHandler(){ ... }
Type:
  • Event

closeHandler

위젯의 닫기 버튼 클릭시 호출됩니다. true가 리턴되면 위젯이 닫힙니다

widget.closeHandler(){ ... }
Type:
  • Event

componentReadyHandler

component를 사용할 준비가 되면 호출합니다.


widget.componentReadyHandler(componentId){ ... }

componentId : 컴포넌트의 아이디

Type:
  • Event

componentRemoveHandler

component가 삭제되기 전 호출됩니다.

widget.componentRemoveHandler(){ ... }
Type:
  • Event

contentReadyHandler

content를 사용할 준비가 되면 호출됩니다.

widget.contentReadyHandler(){ ... }
Type:
  • Event

contentRemoveHandler

content가 삭제되기 전 호출됩니다.

widget.contentRemoveHandler(){ ... }
Type:
  • Event

deactiveHandler

위젯이 소속된 슬라이드가 비활성화되면 호출됩니다.

widget.deactiveHandler(){ ... }
Type:
  • Event

maximizeHandler

위젯의 최대화 버튼 클릭시 호출됩니다.


widget.maximizeHandler(type, contentWidth, contentHeight){ ... }

type : 이벤트 종류. 유효값 : "maximize", "normal"

contentWidth : content영역의 너비.
contentHeight : content영역의 높이.

Type:
  • Event

reload

위젯의 리로드 함수를 호출할 때 호출됩니다.

widget.reload(){ ... }
Type:
  • Event

resizeHandler

위젯의 리사이저로 위젯의 크기를 조정할 때 호출됩니다.


widget.resizeHandler(width, height){ ... }

width : 위젯의 너비.
height : 위젯의 높이.

Type:
  • Event
