Class: MapSeries

MapSeries

new MapSeries()

지도를 표시하는 컴포넌트입니다.

Methods

addEventListener(type, listener)

type에 선언된 이벤트가 발생할 경우 이벤트를 받을 수 있는 listener함수를 등록합니다.

Name Type Description
type string

이벤트 종류를 표현하는 문자열
listener function

이벤트 발생시 불려질 함수

removeEventListener(type, listener)

type에 선언된 이벤트에 연결된 함수 listener를 제거합니다.

Name Type Description
type string

이벤트 종류를 표현하는 문자열
listener function

이벤트 발생시 불려질 함수

Styles

alphanumber

투명도(alpha) 값을 결정합니다. 유효값:0~1

Default Value:
  • 1

areaCodeFieldstring

맵데이타에서 적용할 지역코드 필드명

Default Value:
  • code

blinkIntervalnumber

선택된(클릭된) 지역 이미지의 깜빡거림 시간 간격(밀리세컨드)

Default Value:
  • 400

colorrgb

텍스트의 색깔을 나타냅니다.

disabledColorrgb

잠긴상태의 텍스트 컬러를 나타냅니다.

disabledFillrgb

사용자 선택 불가 지역 이미지의 색상

disabledStrokergb

사용자 선택 불가 지역 이미지의 경계 라인 색상

enabledFieldstring

지역 이미지의 사용자 선택가능여부를 지정하는 data provider의 필드명

filtersarray

시리즈에 적용할 필터를 설정합니다. DropShadowFilter, GlowFilter가 가능하며 여러 개를 동시에 적용할 수 있습니다.

fontAlphanumber

폰트 투명도를 설정합니다.

fontFamilystring

사용할 폰트를 결정합니다.

fontSizenumber

폰트 사이즈를 픽셀단위로 결정합니다.

fontStylestring

폰트 이텔렉체를 사용할지를 나타냅니다. 유효값: "normal","italic"

fontWeightstring

폰트 볼드체를 사용할지를 나타냅니다. 유효값: "normal","bold"

goUpButtonOnlyboolean

DrillDown 하위 맵에서 버튼으로만 상위맵으로 이동할 수 있도록 지정 여부

Default Value:
  • false

hideOverSizeLabelboolean

라벨의 크기가 지역이미지보다 큰 경우 라벨을 숨길지 여부

Default Value:
  • true

interactiveboolean

사용자 마우스 조작에 반응할지 여부. false일 경우 마우스 롤오버나 클릭이 안되며 disabledFill색상으로 표시됩니다.

Default Value:
  • true

labelCallOutboolean

label callOut 사용 유무

Default Value:
  • false

labelCallOutColorrgb

label callOut 라벨 색상

Default Value:
  • #000000

labelCallOutCurveGapnumber

콜아웃 라벨이 꺾이는 거리

Default Value:
  • 5

labelCallOutLineColorrgb

label callOut 라인 색상

Default Value:
  • #000000

labelCallOutLineWidthnumber

콜아웃 정렬 정책. 유효값: "top", "bottom", "middle", "near"

Default Value:
  • 1

labelCallOutPolicystring

label callOut 라인 굵기

Default Value:
  • "middle"

labelColorFieldstring

라벨의 색상을 표현하는 데이터 필드값

labelFieldstring

지역 이미지의 라벨을 지정하는 data provider의 필드명

labelFiltersarray

시리즈의 라벨에 적용할 필터를 설정합니다. DropShadowFilter, GlowFilter가 가능하며 여러 개를 동시에 적용할 수 있습니다.

labelPositionstring

라벨의 위치를 설정. none일 경우 라벨을 표시하지 않습니다. 유효값: "none","inside"

Default Value:
  • "none"

localFillrgb

지역 이미지 기본 색상 - 설정하지 않을 경우 SVG의 색상이 사용됩니다.

localFillByRangearray

최대 최소 값에 맞춰진 색상 범위 값 설정

mapColorFieldstring

맵의 색상을 표현하는 데이터 필드값

patternstring

지역에 표시할 패턴을 지정합니다. 패턴은 BIX5/Assets/Patterns 폴더에 있습니다. 폴더안의 이미지명을 입력하면 됩니다.

localFill을 사용하게 될 경우 localFill 색상으로 덮어집니다.

rangeLegendDataFieldstring

localFillByRange의 기준 값이 되는 데이터 필드값

rollOverFillrgb

마우스 오버된 지역 이미지의 색상

rollOverFilterboolean

롤오버 시 필터 적용 유무

Default Value:
  • false

rollOverFontColorrgb

마우스 오버시 폰트 색상

rollOverStrokergb

마우스 오버된 지역 이미지의 경계 라인 색상

selectionFillrgb

선택된(클릭된) 지역 이미지 색상 - selectionMarking가 color일 경우

selectionMarkingstring

이미지를 클릭 할 경우 해당 이미지에 표시하는 방식 지정. "line", "blink", "color", "none"이 가능하며 "line"의 경우에는 빗금으로 표시하고,

"blink"의 경우 선택된 지역이 깜빡이며, "color"의 경우 스타일의 selectionFill속성의 컬러를 표시하며, "none"의 경우에는 아무런 표시를 하지 않습니다.

Default Value:
  • "color"

selectionMarkLineColorrgb

선택된(클릭된) 지역 이미지 라인 표시 색상 - selectionMarking가 line일 경우

selectionStrokergb

선택된(클릭된) 지역 경계 라인 색상

selectionStrokeAlphanumber

선택된 맵의 Stroke 투명도

selectionStrokeSizestring

selectionMarking이 line인 경우 선의 간격. 유효값 : "small", "large"

showDataEffecteffect

지역 이미지가 표시될때 적용할 이펙트

showLabelFieldstring

라벨유무를 표현하는 데이터 필드값 (데이터는 true/false 로 구분)

Strokergb

지역 이미지 경계 라인 색상

styleNamestring

스타일 이름으로 스타일 지정

textDecorationstring

폰트 underline 지정. 유효값: "underline","none"

toolTipAlphanumber

툴팁 배경 투명도

toolTipBorderColorrgb

툴팁 프레임의 선 색상

toolTipColorrgb

툴팁 글자색상

toolTipFillrgb

툴팁 배경색상

transparentValuenumber

rollOverFill="transparent" 일경우 마우스 오버시하이라이트를 주는 속성. 유효값: 0~100

useGISboolean

GIS 데이터 사용 유무

Default Value:
  • false

useHandCursorboolean

마우스 오버시 손모양 커서 표시 여부

Default Value:
  • false

visibleboolean

가시화 할지 여부를 나타냅니다.

Default Value:
  • true

Events

click

사용자가 마우스를 클릭하면 발생합니다.

altKey : alter 키 클릭여부
controlKey : 컨트롤 키 클릭여부
ctrlKey : 컨트롤 키 클릭여부 (Mac일 경우)
shiftKey : Shift 키 클릭여부

clientY : Y 좌표

  • Event

controlKey : 컨트롤 키 클릭여부
ctrlKey : 컨트롤 키 클릭여부 (Mac일 경우)

clientX : X 좌표
clientY : Y 좌표

Type:
  • Event
